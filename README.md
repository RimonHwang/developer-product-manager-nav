## :question: &nbsp;What is it?

互联网相关的**有趣且立体**的站点分享，包括但不限于个人博客、微信公众号、微博、知乎和网站。

欢迎star和pull request。

## :woman_technologist: &nbsp;For whom?

互联网人，目前主要面向Android开发和产品经理。

## :dart: &nbsp;Goal?

互联网新人获取信息的第一站。
